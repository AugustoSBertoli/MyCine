# 🎬 MyCine

Aplicação web de descoberta e catalogação de filmes, desenvolvida com HTML, CSS e JavaScript puro, consumindo a API do TMDB (The Movie Database).

## 🖥️ Páginas

- **Login / Cadastro** — autenticação local com localStorage, seleção de avatar
- **Home** — listagem de filmes populares, busca por título, filtro por gênero e scroll infinito
- **Detalhes** — sinopse, elenco, avaliação, onde assistir no Brasil e filmes similares
- **Perfil** — filmes assistidos, favoritos e listas personalizadas
- **Lista** — gerenciamento de listas de filmes criadas pelo usuário

## 🚀 Tecnologias

- HTML5
- CSS3 (variáveis CSS, Flexbox, Grid, responsivo)
- JavaScript (Fetch API, async/await, localStorage, URLSearchParams)
- [TMDB API](https://www.themoviedb.org/documentation/api)
